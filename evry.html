Commands

Role Commands

!createrole <name> — Creates a new role.

!deleterole <name> — Deletes a role.

!giverole @user <role> — Gives a role to a user.

!removerole @user <role> — Removes a role from a user.

Utility Commands

!write #channel message — Sends a message as the bot.

!ping — Checks bot latency.

Roblox Linking

!link <game-url> — Links your Roblox game to the bot.

Roblox Linking Tutorial

Step 1 — Use the Link Command

In your Discord server, type:

!link <your-game-api-url>

Step 2 — Add the API to Your Roblox Game

Inside your Roblox game, create a script that sends data to the URL you linked. This allows your game to communicate with the bot.

Step 3 — Test the Connection

Trigger an event in your Roblox game and check if the bot responds.
